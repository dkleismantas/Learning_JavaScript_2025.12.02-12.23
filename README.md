# JavaScript learning course

## Overview

In this course, I learned the basics of JavaScript. I became familiar with variables, logical operators, loops, functions, arrays, and objects. We also briefly had one lecture about the DOM.

# Technologies used

- JavaScript
- Microsoft Visual Studio Code
- Node.js
- Online tools

# How to run the assignments

**Warning:** Some assignments are provided in Lithuanian.

- Clone the repository. Inside, you will find topics and the tasks within them. Each topic contains one or several assignments.
- When you open a task folder, you will find two files with the extensions .js and .txt.
- There are two ways to run an assignment.

## Method 1 *(Recommended, because some solutions use the browser’s alert window, which VS Code cannot display)*

- Open the **.txt** file with Notepad or the **.js** file with VS Code.
- Inside, you will find the tasks, which are commented out and therefore not executed as code, as well as the solution for each task written below it, which is not commented out.
- Copy the solution of the task and paste it **[here](https://jsbin.com/?js,console)** or **[here](https://playcode.io/empty-javascript)** (In the first link, you must press Run to execute the code; in the second link, the code runs automatically.)
- The result will appear in the console window.

## Method 2 *(Less suitable, because some tasks require browser functionality. Do not use this method for tasks that contain the alert or prompt methods)*

- Download and install **Node.js**.
- Open **Microsoft Visual Studio Code**.
- Open the chosen task folder in **VS Code**.
- Open the **.js** file.
- Comment out all other solutions except the one you want to run.
- On the left side of **VS Code**, select **Run and Debug**.
- In the menu that opens, click **Run and Debug** (the first time, you may be asked to choose a debugger — select Node.js).
- A window will appear at the bottom, and the **DEBUG CONSOLE** will show the result.
